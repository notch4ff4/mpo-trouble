# Причины

| Причина                                                  | Описание                                                                                                                                                           | Подтверждено? |
| -------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------- |
| Функция захвата рабочего стола в Nvidia ShadowPlay       | Если запись/повтор включены во время игры, то любой оверлей (*будь-то уведомление с Telegram*) переводит на время игру в `Composed: Flip`                          | Да            |
| "Масштабирование изображение" в панели управление Nvidia | Вызывает `Composed: Flip` в любом случае, подтверждено с помощью замедленной съемки                                                                                | Да            |
| Переключение режима окна                                 | При переключении режима окна с "Полноэкранного" на "Полноэкранный в окне" и без перезапуска игры.<br>\*проявляется не во всех играх, нужен список протестированных | Да            |

Так же в какой-то момент без причины PUBG переключился на Composed: Flip, воспроизвести это не смог.

После включения MSAA игры начинают всегда захватываться через OBS. При Нет или CMAA игры не захватываются в полноэкранно-оконном режиме
